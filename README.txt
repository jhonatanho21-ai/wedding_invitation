WEDDING INVITATION — Jhonatan & María Alejandra
===============================================

Contenido:
- index.html (invitación web completa)
- /images (tus fotos)
- /audio (coloca aquí tu música si quieres usar un archivo local)

Cómo publicar:
1) NETLIFY
   - Arrastra la carpeta 'wedding_invitation' al panel de Netlify Drop (https://app.netlify.com/drop).
   - Obtendrás una URL pública al instante.

2) GITHUB PAGES
   - Crea un repositorio y sube el contenido de esta carpeta.
   - Activa GitHub Pages (Branch: main, /root).

3) HOSTING PROPIO
   - Sube todos los archivos por FTP. Abre index.html en el navegador.

RSVP (Google Forms):
- Edita 'index.html' y reemplaza https://forms.gle/FORM_ID por el enlace real de tu formulario.
- Recomendado: añade campos Nombre, Asistencia (Sí/No), Nº de acompañantes, Mensaje/alergias.

Imágenes:
- Reemplaza las fotos en /images por las tuyas (usa los mismos nombres para no tocar el HTML).

Música:
- Actualmente apunta a una pista gratuita (Pixabay). Si quieres usar un archivo local:
  - coloca un MP3 en /audio, por ejemplo /audio/musica.mp3
  - y cambia la etiqueta <source> del <audio> en index.html a 'audio/musica.mp3'.

Soporte de cuenta regresiva:
